+ **规则**
    + **物体`Path`及其对应名词**
        + 添加。
        + 直接用于`SELECT`物体的移动判定。
        + 可锁定，也可设置解锁条件。

        + 添加。
        + 暂无作用。

    + 部分修复了读取关卡包时错误的物体类型名称会导致游戏尝试持续生成元文本以找到匹配的名称，导致函数调用栈溢出，程序崩溃的Bug。

    + **`refs`**
        + 添加。

    + **`collects`**
        + 添加。

    + **`objects.BmpObject`**

    + **`levelpacks.Levelpack.get_exist_level`**
        + 重命名为`get_exact_level`。
